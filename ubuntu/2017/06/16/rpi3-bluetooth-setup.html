RPi 3: Bluetooth Setup

Jun 16, 17

In order for the Bluetooth to work.. we need to have this file.

  • BCM43430A1.hcd


Make sure this are installed:

  • bluetooth
  • bluez


Put this file: BCM43430A1.hcd in /lib/firmware/brcm folder… then create a symlink of it in /etc/firmware folder.

$ sudo mkdir /etc/firmware
$ sudo ln -s /lib/firmware/brcm/BCM43430A1.hcd /etc/firmware/BCM43430A1.hcd

Then to test it.. run this twice:

$ sudo /usr/bin/hciattach /dev/ttyAMA0 bc43xx 921600 noflow

First run we’ll encouter Initialization timed out. But on the second time we’ll get this Device setup complete. Then try it out now if our device is present.

$ hcitool dev

    Devices:
        hci0   [your device id shows here]

Then run your friendly bluetoothctl. To automate this at boot, just follow the scripts from the reference below.
