WebUI using .Net Core and Bootstrap4: #2

08/28/17

What i used:

  • .Net Core 2 // Latest Release
  • RPi3.Core // customize bcm2835 library. No need for mono to be installed


RPi3.Core:

  • What works are as follows:
  • Set GPIO Output and Input
  • Get GPIO pin level[High/Low]
  • I2C
  • Pull-up/Pull-down
  • You may continue to modiy: based on this post
  • SPI
  • PWM
  • Edge Detection


To reference a .net core project use:

> dotnet add reference RPi3.Core/RPi3.Core.csproj

https://docs.microsoft.com/en-us/dotnet/core/tools/dotnet-add-reference
