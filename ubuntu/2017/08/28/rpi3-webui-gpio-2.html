    RPi3.Core:

    • What works are as follows:
    • Set GPIO Output and Input
    • Get GPIO pin level[High/Low]
    • I2C
    • Pull-up/Pull-down
    • You may continue to modiy:
    • SPI
    • PWM
    • Edge Detection

    To reference a .net core project use:

    > dotnet add reference RPi3.Core/RPi3.Core.csproj
