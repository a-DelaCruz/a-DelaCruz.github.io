RPi 3: Wifi config

Jul 11, 17

Alpine linux network interface configuration are very different from what i used on ubuntu. But wpa_supplicant.conf are still the same. For the interface file:

auto lo
iface lo inet loopback

allow-hotplug eth0
iface eth0 inet dhcp
        hostname [your hostname]

auto wlan0
iface wlan0 inet dhcp
        hostname [your hostname]

And for the wpa_supplicant.conf : (WPA2)

network={
        ssid="your network name"
        psk="your password"
        proto=RSN
        key_mgmt=WPA-PSK
        pairwise=CCMP
        auth_alg=OPEN
        scan_ssid=1     // To be able to connect to hidden network
        mode=0
        group=CCMP
}
